# 📊 Portuguese Bank Marketing Analysis

## 📌 Project Overview
This project focuses on analyzing the Portuguese Bank Marketing dataset to predict whether a client will subscribe to a term deposit. The goal is to extract insights from the data and build machine learning models to support business decision-making.

---

## 🎯 Problem Statement
Banks run marketing campaigns to promote term deposits. However, contacting all customers is inefficient. This project aims to:
- Analyze customer data
- Identify patterns influencing subscription
- Build a predictive model to target potential customers

---

## 📂 Dataset
- Dataset: Portuguese Bank Marketing Dataset
- Contains customer demographic, financial, and campaign-related information

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn
- Jupyter Notebook

---

## 🔍 Exploratory Data Analysis (EDA)

Performed detailed analysis including:
- Data shape, structure, and types
- Duplicate value removal
- Null value check (No missing values found)
- Distribution of categorical and numerical features

### 📊 Key Insights:
- Around **88% of customers did not subscribe**
- Only **~11% subscribed to the term deposit**
- Strong class imbalance in target variable
- Features like job, marital status, and previous outcomes show influence

---

## ⚙️ Data Preprocessing
- Removed duplicate records
- Encoded categorical variables
- Feature selection based on analysis
- Prepared dataset for modeling

---

## 🤖 Machine Learning Models
- Logistic Regression
- Decision Tree Classifier

---

## 📈 Model Evaluation
- Compared model performance using accuracy and other metrics
- Analyzed feature importance
- Observed model behavior on imbalanced data

---

## 📌 Conclusion

This project demonstrates how data analysis and machine learning can help banks improve marketing efficiency.

Key takeaways:
- Customer response is highly imbalanced
- Certain features significantly influence subscription decisions
- Predictive models can help target the right customers and reduce unnecessary calls
